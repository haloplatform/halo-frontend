Dashboard Flexibility and Power

The adjustable dashboard showcases tools and widgets you'll love.

Access Your Favorite Features Fast

The Halo Dashboard is meant to give you instant access to nearly every feature you might want with the flexibility it offers. At the same time it allows you access to your profile settings, security, API keys and account verification and more. With each user comes a particular setup that works best for them, and we want the dashboard to be exactly what you need in the crypto world.

Track every transaction you make for each wallet you've integrated, all in one place. One of our goals was to make things easier for both vets and noobs alike, condensing the expansive universive of crypto into once decentralized control exchange or DCE. Creating an excellent platform for safely and easily integrating only the assets you want, but providing the power to add them all should you want to.

  • Track Portfolio Stats in Multiple Formats.
  • Check Messages and Alerts Fast.
  • Update Profile, Security Info and More Fast.
  • Exchange for Other Coins Effortlessly.
  • Check Recent Transactions Instantly.

Exchange Quickly

We want everyone to be able to quickly exchange their coin safely to the right place, our exchange widget takes care of that.

Enjoy exchanging easily nearly every top coin available, as just one flexible feature of your Halo Dashboard.
